Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

BooksToRead

Bij dit project is het doel om een Angular frontend applicatie te realiseren in combinatie met een NoSQL database. Bij mijn project horen 7 eintiteiten: User, Leeslijst, Boek, Review, Auteur, Uitgever en Adres. Het doel van de applicatie is dat een gebruiker kan aangeven of hij of zij een boek heeft gelezen, deze in een lijst kan zetten en eventueel een review kan maken over het gelezen boek.
