Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

BooksToRead

Bij dit project is het doel om een Angular frontend applicatie te realiseren in combinatie met een NoSQL database. Bij mij project zijn er 7 eintiteiten: User, Leeslijst, Boek, Review, Auteur, Uitgever en Adres.

Entiteiten

Hier geef je een toelichting op je entiteiten. Voeg ook een afbeelding van je model toe zoals dat bij NoSql gevraagd wordt.

User

  • id: number
  • naam: string
  • emailAdres: string
  • phoneNumber: string
  • isStudent: boolean

LeesList

  • id: number
  • naam: string
  • description : string

Review

  • id: number
  • title: string
  • description: string
  • rating: number

Boek

  • id: number
  • title: string
  • summary: string
  • rating: number
  • price: number
  • hasBeenRead: boolean

Auteur

  • id: number
  • firstName: string
  • lastName: string
  • birthDate: date

Publisher

  • id: number
  • name: string
  • founded: date
  • founder: string

Adres

  • id: number
  • postalCode: string
  • street: string
  • houseNumber: number
